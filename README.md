# QuizNiva Website

QuizNiva is an education-first quiz website for beginner, intermediate, and advanced Python notes and MCQ practice.

## Current Slice

- Astro project structure
- Home, subjects, Python subject, topic, quiz, and legal pages
- Thirty complete Python topic pages across beginner, intermediate, and advanced levels
- Thirty complete Python quizzes with 20 questions each
- 600 original MCQs with answer explanations
- Local best-score storage in the browser

## Run Locally

Node.js and npm are required.

```bash
npm install
npm run dev
```

Then open the local URL shown by Astro.

## Build

```bash
npm run build
```
