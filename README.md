# QuizNiva Website

QuizNiva is an education-first quiz website for beginner, intermediate, and advanced Python notes and MCQ practice.

## Production Features

- Astro static site build
- Home, subjects, Python, Computer Fundamentals, topic, quiz, search, sitemap, and legal pages
- Thirty complete Python topic pages across beginner, intermediate, and advanced levels
- Thirty complete Python quizzes with 20 questions each
- 600 original MCQs with answer explanations
- Computer Fundamentals lessons and quizzes
- Local progress and best-score storage in the browser
- XML sitemap, HTML sitemap, canonical tags, schema markup, and ads.txt

## Run Locally

Node.js and npm are required.

```bash
npm install
npm run dev
```

Then open the local URL shown by Astro.

## Build

```bash
npm run build
```
